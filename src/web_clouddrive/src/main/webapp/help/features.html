Here's a short rundown of the features of cloud backed storage:


  • - Disk (WebDAV) interface compatible with most major clients (Win, OS X, Linux, iOS)

  • - Website interface

  • - Tagging in the web interface

  • - Search on tags

  • - Saving searches as folders (and they are kept up to date)

  • - Auto-tagging of files that are dropped on a stored search folder

  • - Executing a search query based on tag names via the disk interface
