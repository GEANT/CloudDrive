Windows 7
On Windows 7 you can just use the built-in file manager (explorer).
Click “create network drive” In the url field type “https://disk.beneathclouds.com”; next give your username and the password you chose when you activated the disk. You should see the disk in the file manager and be good to go!


Windows XP and Windows Vista
On XP and Vista your mileage may vary - there is an endless list of service packs and bug fixes. Windows 7 does not have any of these problems and you may give it a try with the instructions for Windows 7.
Otherwise on XP and Vista the consensus is that the freeware Bitkinex is the best client. Download, install, and create a new WebDAV SSL connection to https://rightfabric.net Use the username and password to connect.
